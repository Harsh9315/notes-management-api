# Daily Notes Management System

A full-stack web application designed to help users log, organize, and manage their daily notes efficiently. This project features a clean, responsive frontend and a Node.js backend.

## 🚀Key Features
- **Daily Logging:** Easily save notes with specific dates.
- **Filter System:** Quickly filter your saved entries by date.
- **Modern UI:** Responsive and user-friendly interface with a modern aesthetic.
- **Backend Integration:** Powered by a Node.js server to handle data requests.

## 🛠 Technologies Used
- **Frontend:** HTML5, CSS3, JavaScript
- **Backend:** Node.js, Express.js (`server.js`)

## ⚙️ How to Run Locally

Navigate to the project folder:
cd <your-folder-name>

Install dependencies:
npm install express

Start the server:
node server.js

Open your browser and go to http://localhost:3000
